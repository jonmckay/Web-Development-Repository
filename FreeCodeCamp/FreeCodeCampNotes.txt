Regular Expressions
--------------------

/./gi

- g means that we want to search the entire string for this pattern instead of just the first match

- i means that we want to ignore the case (uppercase or lowercase when searching for the pattern)

\d  - used to retrieve the numbers in a string

+  - following the digit selector allows this regular expression to match multi-digit numbers

\s - finds whitespace in a string (tabs, spaces, returns)

invert matches by using uppercase version of regular expression (ex: \s would be \S)
